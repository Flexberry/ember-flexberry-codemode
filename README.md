# Ember Flexberry Codemode

Codemods for ember-flexberry.

## Installation

```
npm install -g ember-flexberry-codemode
```

## Running

```
ember-flexberry-codemode flexberry addon/**/*.js app/**/*.js tests/**/*.js
```

## Running Tests

```
codemod-cli test
```
Options:
* --testNamePattern
* --runInBand
